Common Issues

  • Analysis failed or timed out — include the wallet address (or partial address) and approximate time of the error.
  • Report page blank or error — include the full report URL (e.g., /report/...).
  • Rate limit (5 reports/day) — free tier limit per IP; wait 24 hours or describe your use case if you need higher limits.
  • Incorrect or incomplete token data — some wallets with thousands of tokens may show limited holdings due to API constraints; include the wallet address.
  • Share link not working — include the link you copied and your browser/device.

Information to Include

To help us resolve issues faster, please provide:

  1. Description of the problem
  2. Steps to reproduce
  3. Browser and device (e.g., Chrome on iPhone, Safari on Mac)
  4. Screenshot if possible
  5. Wallet address or report URL (if relevant)

What We Cannot Help With

  • Investment or trading advice — see Disclaimer.
  • Recovering lost crypto, private keys, or wallet passwords.
  • Disputes with exchanges, protocols, or other third parties.
  • Guaranteeing report accuracy — outputs are informational only.
